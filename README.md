Suggestion Bot
  <br>
</h1>


## Features

`!suggest`: Suggests a suggestion <br>
`!accept [messageID]` : Accepts a suggestion (Do not spam) <br>
`!decline [messageID]` : Declines a suggestion (Do not spam) <br>
